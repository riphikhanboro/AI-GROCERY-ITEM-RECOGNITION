## Dataset Information

The dataset is currently under preparation and expansion.

For structural demonstration purposes, a small sample has been uploaded to this repository to showcase the folder organization and naming convention.

Due to GitHub repository storage and file size limitations, the complete dataset(images) has not been uploaded here. The final dataset will include multiple packaged and unpackaged grocery items with different sub-types, captured under varied lighting conditions and angles.

The complete dataset will be made available separately once preparation is finalized.
